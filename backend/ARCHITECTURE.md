```

   Body: { url: "https://gateway.lighthouse.storage/ipfs/..." }

   ‚îú‚îÄ CORS middleware ‚úì
   ‚îú‚îÄ Body parser ‚úì
   ‚îî‚îÄ Routes ‚Üì

   ‚îî‚îÄ Forwards to /api/* ‚Üí video.routes.ts

   ‚îî‚îÄ POST /transcode-url ‚Üí transcodeFromUrl()

5. controllers/video.controller.ts
   ‚îú‚îÄ Validate URL ‚úì
   ‚îú‚îÄ Generate videoId
   ‚îú‚îÄ Check cache

   ‚îî‚îÄ HLS output (playlist + segments)
```
